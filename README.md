## Visão Geral

Este é o meu primeiro projeto console em C# do módulo **"Projeto guiado – Calcular GPA final"** da Microsoft Learn (parte introdutória de C#).

O objetivo é calcular a média final (GPA) de alunos com base em suas notas em várias tarefas/avaliações e exibir uma tabela simples com o nome, média e nota conceitual (A/B).

**Link oficial do módulo**:  
[https://learn.microsoft.com/pt-br/training/modules/guided-project-calculate-final-gpa/](https://learn.microsoft.com/pt-br/training/modules/guided-project-calculate-final-gpa/)

## O que aprendi nesse projeto

- Declaração de variáveis (`int` para notas, `decimal` para médias precisas)
- Casting explícito `(decimal)` para evitar perda de precisão na divisão
- Interpolação de strings moderna (`$"..."`) para formatar saída
- Uso de `\t` para alinhar colunas no console
- Cálculo simples de média aritmética
- Lógica básica de decisão (atribuição manual de letras A/B baseado na média)
